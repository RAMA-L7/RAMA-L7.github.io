üß† Foundational Concepts

  • AI (Artificial Intelligence): Teaching machines to act smart, like humans.
  • ML (Machine Learning): Helping machines learn from data (not rules).
  • DL (Deep Learning): A type of ML using brain-like networks.
  • NLP (Natural Language Processing): How AI understands human language.
  • Computer Vision: How AI sees images or videos.
  • Model: The AI brain that makes decisions.

üß™ Machine Learning Terms

  • Training: Teaching a model with example data.
  • Dataset: A big collection of training data.
  • Features: Inputs the model learns from.
  • Labels: Correct answers in training data.
  • Supervised Learning: Learning with labeled data.
  • Unsupervised Learning: Finding patterns without labels.
  • Overfitting: When a model memorizes too much.
  • Underfitting: When a model is too simple.
  • Accuracy / Precision / Recall: Metrics to judge model quality.

üß† Deep Learning & Neural Networks

  • Neuron: A tiny brain-like unit.
  • Layer: A group of neurons.
  • Neural Network: A system of layers that learns patterns.
  • CNN (Convolutional Neural Network): Good for image tasks.
  • RNN (Recurrent Neural Network): Good for sequences like text.
  • Transformer: A model type used in ChatGPT.
  • Embedding: Turning text/images into numbers.
